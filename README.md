## v15 share update

シェア本文に8桁コードが必ず入るように変更しました。

- ひとり診断: 相性コードを本文に含める
- ふたり診断: あなた/相手の2つの8桁コードを本文に含める
- X / LINE / スマホ標準共有 / コピーで、URLだけでなくコード入り本文を共有
