# 夜の16タイプ診断｜アルパカ夜人格 v4

8桁の相性診断コードで、144パターン同士の相性診断ができる版です。

## v4で変更したこと

- トップから「ふたりで相性診断する」を押す
- 利用規約に同意する
- MBTI選択ではなく、2人分の「8桁相性コード」を入力する画面へ遷移
- コード同士で144パターン相性診断を実行
- 結果画面もコード相性用に変更

## 8桁コードの仕様

ひとり診断の結果画面で発行されます。

例：`2F7A9K3P`

コードには以下の情報を含みます。

- 144パターン番号
- 16タイプ情報
- 夜タイプ軸
- チェック用ハッシュ

ふたり相性診断では、このコードを2つ入力して、144パターン同士の距離で相性を算出します。

## 画面フロー

### ひとり診断

トップ → 利用規約 → MBTI登録/12問診断 → 結果 → 8桁コード発行

### ふたり相性診断

トップ → 利用規約 → 8桁コード2つ入力 → 相性結果

## 質問数の変更

質問は `app.js` の `qs` 配列で管理しています。
進捗表示・終了判定・結果計算は `qs.length` を参照しているため、12問から25問へ増やしても基本構造は動きます。
