# 夜の16タイプ診断｜アルパカ夜人格 v5

この版では、ふたり相性診断を **MBTI/16タイプ選択ではなく、8桁の相性コード入力だけ** に変更しています。

## 主な変更点

- トップ → ふたりで相性診断する → 利用規約同意 → 8桁コード入力画面
- ふたり診断画面からMBTI/16タイプ選択を削除
- ふたり診断画面から「12問で仮登録する」を削除
- ひとり診断の結果画面で、144パターン番号を含む8桁コードを発行
- ふたり相性診断は、2つの8桁コードから144パターン同士の距離で判定
- 質問数は `qs.length` ベースなので、12問から25問に増やしても進捗・終了判定が崩れにくい設計

## アップロード方法

既存のGitHubリポジトリに、このフォルダ内のファイルをすべて上書きアップロードしてください。

- `index.html`
- `style.css`
- `app.js`
- `terms.txt`
- `alpaca-promo.png`

GitHub連携済みのVercelは、GitHubへの反映後に自動で再Deployされます。
