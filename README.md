## 画像管理

画像パスは `character-assets.js` で管理しています。

```js
window.characterAssets = {
  hero: 'assets/paka-ne-main.png',
  head: 'assets/paka-ne-head.png',
  guide: 'assets/paka-ne-main.png',
  pair: 'assets/paka-ne-main.png',
  result: 'assets/paka-ne-main.png',
  deep: 'assets/paka-ne-main.png'
};
```

今後キャラを差し替える場合は、基本的に `assets/` 内の画像を差し替えるか、`character-assets.js` のパスを変えるだけで済みます。
