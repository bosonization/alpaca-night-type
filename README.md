# 夜の16タイプ診断｜アルパカ夜人格 v2

追加済み機能：

1. トップから診断開始時に利用規約画面を表示
2. 同意チェック後に診断開始
3. 16タイプ表示に加えて、144パターン相性診断用の8桁コードを発行
4. 144パターンコードを使った相性スコア計算
5. 質問数を増やしても動く可変設計

## ファイル構成

- `index.html`
- `style.css`
- `app.js`
- `alpaca-promo.png`

## 質問を増やす方法

`app.js` の `qs` 配列に質問を追加してください。
画面進捗・終了判定・仮タイプ判定は `qs.length` を参照しているため、12問から25問に増やしても動く設計です。

## 144パターンコード

`patternInfo(type, ans)` で生成しています。

- 12×12 = 144 の内部パターン番号
- 8桁の英数字コード
- 相性診断では2人のX/Yパターン距離を使ってスコア化

## 合言葉の変更

`app.js` の先頭を変更します。

```js
const MONTHLY_PASSWORD='午前2時';
```

ユーザー提供の正式な利用規約全文を、診断開始前の同意画面に組み込みました。

- 確認用テキスト: `terms.txt`
